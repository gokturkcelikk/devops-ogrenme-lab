DevOps öğrenme sürecimde yaptığım Docker, Kubernetes, Terraform ve Linux çalışmalarını içerir.
# DevOps Öğrenme Lab

Bu repo, DevOps alanında kendimi geliştirme sürecimde yaptığım çalışmaları içermektedir.

Amaç:
- Linux sistem mantığını kavramak
- Docker ile container mantığını öğrenmek
- Kubernetes ile uygulama çalıştırma ve yönetim sürecini anlamak
- Terraform ile temel altyapı tanımlama pratiği yapmak

Bu repodaki içerikler:
- Docker temel kullanım örnekleri
- Kubernetes komutları ve basit deployment denemeleri
- Terraform ile temel altyapı tanımları
- Linux komutları ve sistem notları

Not:  
Bu repo öğrenme sürecini belgelemek amacıyla oluşturulmuştur.
